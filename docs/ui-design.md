# UI 디자인 문서 — HUD & 채팅

> 이 문서는 web3DGame의 UI/HUD 레이아웃과 스타일 스펙을 정의합니다.
> 주요 구현 파일: `client/src/ui/hud.ts`, `client/src/ui/chat.ts`, `client/src/game/player.ts`

---

## 1. 화면 레이아웃 구조

```
┌────────────────────────────────────────────────────────┐
│  [상태 패널]                        [조작 가이드]       │  ← top
│  MULTIPLAYER SECURE LINK            CONTROLS           │
│  STATUS: ONLINE                     WASD : 이동        │
│  CONTROLS: W,A,S,D                  ↑↓←→: 카메라      │
│                                     Space: 점프        │
│                                     F/클릭: 발사       │
│                                     Enter: 채팅        │
│                                                        │
│                                                        │
│                   [크로스헤어 +]                        │  ← center
│                                                        │
│                                                        │
│  [채팅 창]                                             │  ← bottom-left
│  ┌──────────────────┐                                  │
│  │ 메시지 기록 영역  │                                  │
│  │                  │                                  │
│  ├──────────────────┤                                  │
│  │ 입력창...         │                                  │
│  └──────────────────┘                                  │
│                                                        │
│              [HP 바]                                   │  ← bottom-center
└────────────────────────────────────────────────────────┘
```

## 2. UI 컴포넌트 상세

### 2-1. 상태 패널 (Status Panel)

| 항목 | 값 |
|---|---|
| 위치 | 좌측 하단 (bottom: 20px, left: 20px) |
| 크기 | 자동 (패딩 15px) |
| 배경 | `rgba(20, 20, 30, 0.4)` + `blur(8px)` |
| 테두리 | 좌측 4px solid `#FF7EDB` (핑크) |
| 모서리 | 0 12px 12px 0 (왼쪽만 직각) |
| 폰트 | monospace, 14px |
| 텍스트 색 | `#E0E0FF` (연보라) |
| 상태 표시 | ONLINE → `#20FFC9` (민트) |

### 2-2. 크로스헤어 (Crosshair)

| 항목 | 값 |
|---|---|
| 위치 | 화면 정중앙 (transform: translate(-50%, -50%)) |
| 기호 | `+` |
| 색상 | `rgba(255, 255, 255, 0.5)` |
| 크기 | 24px |
| 클릭 방해 | 없음 (pointer-events: none) |

### 2-3. 채팅 창 (Chat Window)

| 항목 | 값 |
|---|---|
| 위치 | 좌측 하단 (bottom: 120px, left: 20px) |
| 크기 | 300 × 200px |
| 배경 | `linear-gradient(135deg, rgba(15,15,30,0.6), rgba(30,20,50,0.4))` |
| 테두리 | 1px solid `rgba(255,255,255,0.15)` |
| 블러 | `blur(8px)` |
| 모서리 | 5px |

**메시지 영역:**
- flex: 1, overflowY: auto
- 폰트: monospace, 12px
- 색상: `#E0E0FF`
- 발신자 표시: `[이름]` 최대 12자, 플레이어 컬러

**입력창:**
- 배경: 투명
- 커서 색: `#20FFC9` (민트)
- placeholder: `Press Enter to jack in...`
- placeholder 색: `rgba(224, 224, 255, 0.6)`
- 전송: Enter 키
- 전송 후 blur (포커스 해제)
- 전역 Enter 키로 입력창 활성화

**IME 처리:** `e.isComposing` 체크로 한글 조합 중 이중 전송 방지

### 2-4. HP 바 (Health Bar)

| 항목 | 값 |
|---|---|
| 위치 | 하단 중앙 (bottom: 30px, left: 50%, translateX(-50%)) |
| 크기 | 250 × 24px |
| 외곽선 | 2px solid `#FFFFFF` |
| 모서리 | 12px (pill shape) |
| 배경 | `rgba(0, 0, 0, 0.5)` |
| 애니메이션 | width/color transition 0.3s ease |
| 텍스트 | `HP: {값}`, 흰색, bold, 14px |

**색상 단계:**

| 범위 | 색상 | Hex |
|---|---|---|
| 60 ~ 100 | 초록 | `#6BCB77` |
| 30 ~ 59 | 노랑 | `#FFD93D` |
| 0 ~ 29 | 빨강 | `#FF4D4D` |

### 2-5. 조작 가이드 (Controls Hint)

| 항목 | 값 |
|---|---|
| 위치 | 우측 상단 (top: 20px, right: 20px) |
| 배경 | `rgba(255, 255, 255, 0.7)` |
| 테두리 | 1px solid `#CCCCCC` |
| 모서리 | 5px |
| 폰트 | monospace, 12px |
| 텍스트 색 | `#333` |
| 줄 간격 | 1.8 |

## 3. 전체 색상 팔레트 (UI)

| 용도 | Hex | 설명 |
|---|---|---|
| 기본 텍스트 | `#E0E0FF` | 연보라 (사이버펑크 톤) |
| 강조 (온라인) | `#20FFC9` | 민트/시안 |
| 강조 (테두리) | `#FF7EDB` | 핑크 |
| HP 안전 | `#6BCB77` | 초록 |
| HP 경고 | `#FFD93D` | 노랑 |
| HP 위험 | `#FF4D4D` | 빨강 |
| 파티클 | `#FFAA00` | 주황 |

## 4. 네임태그 (Name Tag)

- 구현: `client/src/game/nameTag.ts`
- 위치: 캐릭터 머리 위 (꽃 위쪽)
- Billboard: 항상 카메라 방향을 향함
- 스타일: 그래피티 폰트 느낌
- 최대 글자: 12자
