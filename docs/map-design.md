## 1. 현재 맵 — "Pastel Garden" (구현 완료)

### 1-1. 환경 스펙

| 항목 | 값 |
|---|---|
| 바닥 크기 | 400 × 400 유닛 |
| 플레이 가능 영역 | 약 160 × 160 (중앙 기준 ±80) |
| 배경 색상 | 하늘색 `#A2D2FF` |
| 안개 | FogExp2, 밀도 0.005 |

### 1-2. 조명

| 조명 종류 | 색상 | 강도 | 비고 |
|---|---|---|---|
| HemisphereLight | `#FFFFFF` / `#B9F3FC` | 1.0 | 환경광 (하늘/땅) |
| DirectionalLight | `#FFFFFF` | 1.8 | 그림자 생성 |

- 태양 방향: position (50, 100, -50)
- 그림자 맵: 4096 × 4096 (고품질)
- 그림자 범위: ±100 유닛

### 1-3. 바닥 (Floor)

- 재질: 절차적 잔디 텍스처 (Canvas 생성)
- 베이스 컬러: `#5A9E3A` (초록)
- 텍스처 타일: 40 × 40 반복
- 랜덤 밝기 점 8,000개로 자연스러운 잔디 표현
- 탄환 충돌 대상 포함

### 1-4. 장애물 (Obstacles)

- 수량: **80개**
- 생성 방식: 시드 기반 난수 (seed=42, 모든 클라이언트 동일)
- 위치 범위: x, z ∈ [-80, +80]

| 속성 | 범위 |
|---|---|
| 높이 | 2 ~ 17 유닛 |
| 너비 | 1 ~ 4 유닛 |
| 깊이 | 1 ~ 4 유닛 |

**파스텔 컬러 팔레트 (장애물):**

| 컬러명 | Hex |
|---|---|
| Salmon | `#FFADAD` |
| Peach | `#FFD6A5` |
| Yellow | `#FDFFB6` |
| Mint | `#CAFFBF` |
| Cyan | `#9BF6FF` |
| Sky | `#A0C4FF` |
| Lavender | `#BDB2FF` |
| Pink | `#FFC6FF` |

- roughness: 0.4, metalness: 0.0
- 탄환 충돌, 그림자 캐스팅 모두 활성화

## 2. 맵 설계 원칙

### 레이아웃 원칙
- **중앙 오픈 공간**: 플레이어 간 교전 가능한 넓은 개방 구역
- **장애물 클러스터**: 은폐/엄폐 가능한 밀집 구역
- **시드 기반 생성**: 서버 없이도 모든 클라이언트가 동일한 맵 렌더링
- **탄환 충돌 등록**: 모든 장애물은 `worldCollidables`에 추가

### 비주얼 원칙
- 파스텔 톤 통일 (캐릭터 컬러와 동일 팔레트)
- 밝고 따뜻한 낮 분위기 유지
- 과도한 어두운 그림자 지양

## 4. 기술 참고

### 현재 구조

```typescript
// world.ts
export const worldCollidables: THREE.Object3D[] = [];

export const initWorld = () => {
  // 배경/안개 설정
  scene.background = new THREE.Color(0xA2D2FF);
  scene.fog = new THREE.FogExp2(0xA2D2FF, 0.005);

  // 조명
  // ...

  // 바닥 (충돌 등록 포함)
  worldCollidables.push(floor);

  // 장애물 (시드 기반, 충돌 등록)
  worldCollidables.push(box);
};
```

### 새 오브젝트 추가 시 체크리스트

```
1. scene.add(mesh) — 씬에 추가
2. worldCollidables.push(mesh) — 탄환 충돌 대상 등록 (필요 시)
3. mesh.castShadow = true — 그림자 캐스팅
4. mesh.receiveShadow = true — 그림자 수신
5. 시드 기반 랜덤 사용 — seededRandom() 함수 활용 (동기화 필수)
```

### 맵 좌표 기준

```
월드 크기:  400 × 400
플레이 존:  ±80 (x, z)
높이:       y = 0 (바닥)
장애물 높이: y = height/2 (바닥에서 솟아오름)
카메라 높이: ~15 유닛 위
```
