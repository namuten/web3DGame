---
// turbo-all

1. 게임 서버 실행 (Port 3000)
   - `cd server && npm run dev` 명령을 터미널에서 실행하여 백그라운드 서버를 구동합니다.

2. 게임 클라이언트 실행 (Port 80 또는 기본 포트)
   - `cd client && npm run dev` 명령을 터미널에서 실행하여 클라이언트 개발 환경을 실행합니다.

3. 어드민 페이지 실행 (Port 81 또는 기본 포트)
   - `cd admin && npm run dev` 명령을 터미널에서 실행하여 관리자 페이지 개발 서버를 구동합니다.
