# 📝 세션 작업 요약 (2026-03-17)

오늘 세션에서 진행된 Three.js 멀티플레이어 게임 개발 작업 요약입니다. 다음에 이어서 작업할 때 참고하세요.

## 🕹️ 개발 완료된 주요 기능
1.  **3D 렌더링 & 월드**:
    *   Three.js 기반의 네온 그리드 바닥과 랜덤 장애물(사이버펑크 스타일) 구축.
    *   안개(Fog) 및 실시간 그림자(Shadow) 적용.
2.  **멀티플레이어 시스템**:
    *   Socket.IO 서버를 통해 다수 플레이어 접속 및 위치 실시간 동기화.
    *   다른 플레이어는 빨간색 캡슐 모델로 표시.
3.  **조작성 개선**:
    *   `WASD`: 이동 (카메라 시점 기준 상대적 이동 지원).
    *   `방향키(↑↓←→)`: 부드러운 오비탈 카메라 회전.
    *   `Space`: 중력 적용된 점프 기능.
4.  **채팅 시스템**:
    *   한글 입력(IME) 시 발생하는 중복 전송 버그 해결.
    *   `Enter` 키로 포커스 전환 및 메시지 전송.
5.  **전투 시스템 (Bullet)**:
    *   `F` 키 또는 **마우스 좌클릭**으로 탄환 발사.
    *   **1인 1발 제한**: 탄환이 날아가는 동안 추가 발사 방지 (아케이드 밸런스).
    *   Raycast 기반 충돌 판정 및 화려한 파티클 폭발 이펙트 적용.
    *   탄환 자체 광원(PointLight)으로 야간 시인성 확보.

## 🚀 실행 방법
터미널 두 개를 열어 각각 실행해야 합니다.

*   **Server**: `cd server && npx ts-node src/index.ts`
*   **Client**: `cd client && npm run dev`
